Overview 
This program is a application that allows the user to easily add expenses by category and show the user how much money they spent in total and per category. 

Features
-add expenses: user can add new expenses with the name, amount, and category
-view all expenses: displays the recorded expenses with dates
-filter by category: this allows the user to filter the expense by category 
-calculate totals: this will calculate the total dollar amount spent for both the category and total of all categorys.
