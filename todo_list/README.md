Todo list application

a simple to do list orginizer that lets you manage tasks efficiently.

features:
-add new tasks to your list
-mark tasks as completed
-delete tasks you no longer need
-view all tasks with there status either completed or pending
-data automatically saves to the JSON file

how to use:
-run 'python3 main.py' to start program 
-choose a option listed from the menu 
-follow the prompts to add a task, mark a task as complete, delete a task, exit program

files include:
-tasks.py: contains all functions
-main.py: is the menu interface for the user
-tasks.json: stores your tasks
