OVERVIEW
this is a basic number guessing game where the program randomly selects a number, the number is ranged based off the difficulty the user selects. the user will guess until the correct number is found and the attempts are counted each time the user guesses a number. after the user guesses the correct number the program will congragulate the user and inform how many attempts it took to guess. 

FEATURES
there are 3 difficulty levels (easy, medium and hard) when the user selects easy the random number will be between 1-100 medium 1-300 hard 1-1000. each attempt guessed the program will inform the user if he is to high or to low and add 1 to the attempts.
